# Interactive Hydrogen Infrastructure Planning Tool

Interactive web-based companion to the paper "Multi-Criteria Hydrogen Infrastructure Planning for Industry and Freight Decarbonization" by Ghorbanali Zadegan & Guo (2026), submitted to *Applied Energy*. The tool implements the multi-criteria optimization framework for the nine-state Northeastern United States, allowing users to visualize the cost-only and multi-criteria configurations on an interactive map, compare freight flows and infrastructure layouts, and examine alternative county-exclusion scenarios.


## What it does

The tool lets you explore alternative hydrogen infrastructure configurations across the nine-state Northeastern United States (Connecticut, Maine, Massachusetts, New Hampshire, Rhode Island, Vermont, New York, New Jersey, and Pennsylvania). You can:

- Visualize the cost-only and multi-criteria solves on an interactive map
- Compare electrolyzer siting, HRS placement, and freight flow patterns
- Examine the cost decomposition of each configuration
- Explore county-exclusion scenarios

## Methodology

The underlying optimization model, data sources, and spatial criteria are documented in the paper. The tool itself runs entirely in the browser using pre-computed solve results embedded in the page; no installation or backend is required.
